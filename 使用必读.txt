使用必读 - 获取免费 API Key
=====================================

使用声明
--------
本项目仅供研究和学习使用。严禁用于非法获取信息、干扰系统或任何违法活动。请仔细审阅使用条款（resources/privacy_policy.txt），在符合法律法规和应用平台规则的前提下使用本工具。

本指南将帮助您从 ModelScope 获取免费的 API Key 来使用 Open-AutoGLM。

步骤 1: 访问模型页面
-------------------
访问以下链接：
https://modelscope.cn/models/ZhipuAI/AutoGLM-Phone-9B

步骤 2: 创建免费实例
-------------------
1. 点击页面上的 "Notebook快速开发" 按钮
2. 选择 "免费创建实例"

步骤 3: 配置实例
-------------------
1. 请选择 "启动实例" -> "免费实例环境"
2. 选择 "PAI-DSW"
3. 选择 "长期使用"
4. 点击 "创建"

步骤 4: 启动实例
-------------------
创建成功后，点击 "启动" 按钮启动实例。

步骤 5: 获取 API Key
-------------------
1. 返回到模型界面（https://modelscope.cn/models/ZhipuAI/AutoGLM-Phone-9B）
2. 点击右侧的 "推理 API-Inference"
3. 查看代码范例：
   - 如果代码中已经有完整的 `api_key='ms-xxxxxx'`，可以直接复制使用
   - 如果没有，点击 "在 Notebook 中打开" 按钮，在 Notebook 中运行代码即可获取 API Key

步骤 6: 在 GUI 中使用
-------------------
1. 启动 GUI 应用：打开GUI
2. 在 "模型配置" 面板中：
   - 预设配置：选择 "智谱-Phone"
   - API Key：粘贴您获取的 API Key（格式：ms-xxxxxx）
3. 点击 "保存配置" 保存设置

注意事项
--------
- API Key 是私密信息，请妥善保管，不要分享给他人
- 免费实例有使用限制，请合理使用
- 如果 API Key 过期，请按照上述步骤重新获取

更多帮助

如有问题，请参考项目 README.md 文件或访问项目 GitHub 页面。
